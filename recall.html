Hexadecimal or hex code:

 #000000
0-9 and A-F are the 16 symbols that can be used in hex code. More than 16M unique colors can be created using the combination of RGB in hex code.

RGB values:

rgb(0,0,0)
Using rgb to select color we can get exactly the same number of distinct color combinations. Since we can select values from 0-255 for each red, green and blue where zero is the lightest and 255 is brightest we get a total of over 16M distinct colors.

Variables:

Using variables we can easily change properties of many different elements simultaneously. Here the value of the variable is set to 50px and has been applied to the padding property of the current h3 element. A fallback value can be set in case the variable is entered incorrectly. For eg var(--bluee-color, blue); the value blue will still be applied this way even when the variable is typed incorrectly.

This will not solve the compatibility issues with older browsers though, you will need to add a separate line and declare the value before choosing to add the variable.

Inheritence works the same way with variables, it will only work inside the selector in which it is defined and its descendant elements. To make a variable accessible by all elements you will need to define it under the pseudo-class ":root".

The variable value which is in the root selector can be overwritten by re defining it under a different selector. This is useful when you need to set an exception for the variable defined under the root class.

Media query can be used to change the variable's value according to the screen size set in the media query break.
